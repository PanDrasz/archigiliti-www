# Brand & UI — Archigiliti

## 1. Osobowość marki
- Ekspercka
- Spokojna
- Techniczna
- Partnerska
- Bez nadęcia

## 2. Styl wizualny
- Minimalizm
- Dużo „oddechu”
- Czytelna typografia
- Stonowane kolory
- Estetyka enterprise / consulting

## 3. Kolory
- **Primary:** #D97706 (złoto/amber — główny kolor marki)
- **Primary dark:** #B45309 (ciemniejszy amber — hover, akcenty)
- **Primary light:** #FCD34D (jasny amber — subtelne tła)
- **Neutral dark:** #1C1917 (stone-900 — tekst, nagłówki)
- **Neutral mid:** #57534E (stone-600 — tekst pomocniczy)
- **Neutral light:** #FAFAF9 (stone-50 — tło)
- **White:** #FFFFFF

## 4. Typografia
- Headings: Inter / SF Pro / Geist
- Body: Inter / System UI
- Brak fontów „ozdobnych”

## 5. Elementy stałe
- Header:
  - logo
  - menu (Home, Oferta, O nas, Kontakt)
- Footer:
  - krótki opis
  - dane kontaktowe
  - copyright
- CTA:
  - proste, wyraźne
  - bez agresywnych kolorów

## 6. Ilustracje / grafika
- Preferowane:
